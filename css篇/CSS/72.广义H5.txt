网站TDK 三大标签 SEO 优化
提高自然排名
SEO 搜索引擎优化（serach engine optimization）
帮助网站获得免费流量

结合TDK 三大标签 title description keyword
title 是网站标题
网站名（产品名）-网站的介绍(尽量不要超过30个字)

description  主要说明网站是做什么的
<meta name = 'description' content="内容"> 

keyword 关键字  是页面的关键词 是搜索引擎的关注点之一
<meta name='keyword' content="关键词1,关键词2">
