# 程序
- 程序是操作系统可以认识的语言，语言经过一个解释后转变为一个操作系统可以识别的程序。
- 程序是被动的，他不会自己把自己加载到内存上。
##### 磁盘
- 容量大
- CPU访问慢
- 不需要电也可以储存数据
##### 内存
- 必须要靠电支持
- 内存是一个cpu与磁盘的中转

# 进程
- 运行程序后 操作系统将程序放在内存里面称为进程
- 程序是静态的，进程是动态的
- 一个程序可以有很多进程
- 程序的代码被加载到内存上让他可以被cpu执行，这样的状态叫做进程

- 用于cpu快速切换进程的运行，造成一种似乎是并行的过程。
